# prime-router
 Routing
 
## Command Options

```
prime [Configure][Input][Action][Output] 

--help

Configure
--receivers_catalog=<catalog_file> - Extend national file with this configuration
--schema_catalog=<catalog_file> - Extend national file with this configuration

Input
--input=<file1>, <file2> 
--input_dir=<directory>
--input_schema=<schema_name>, <schema_name>

Actions
--validate
--merge_on=<col1>, <col2>
--route
--partition_by=<col>

Output
--output=<file>
--output_dir=<directory>
--output_schema=<schema_name>
--send
```
